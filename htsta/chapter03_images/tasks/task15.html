Task 15 - Favicon

The favicon is the small icon that is displayed in a browser tab next to the title of the page. At first glance this is a small detail, but the favicon is a good example of the kind of chaos can arise if new features are not standardized.

This page gives an overview of all cases to be considered (at the time of writing).

For simplicity, we'll focus only on desktop browsers. Research how to add this image a favicon to your solution from Task 14.
