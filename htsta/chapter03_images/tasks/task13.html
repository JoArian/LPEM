Task 13 - Embedding images in HTML

The img element defines an image in HTML. This element does not require any content, but defines the image using the following 2 attributes:

  • src gets as value the path (URL) to the image
  • alt receives as value a very short description of the image (see here).

Usually the images of a web page get their own folder. For this reason, in Task 12 we also saved all scaled images in one folder:

../solutions/images/

To do this, open the file ../solutions/task13.html and paste the scaled images under the appropriate headings. Also use your knowledge regarding relative paths from the previous chapter to embed the images.
