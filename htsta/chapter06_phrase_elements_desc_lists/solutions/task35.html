Computer code and formulas

To determine the IP address of the host name www.lpem.lu you can execute the following command in the command line:

nslookup www.lpem.lu

The output then shows us the corresponding IP address: 

		  
		Server:  snoopy.ltpemdomain.local
		Address:  192.168.100.10
		
		Non-authoritative answer:
		Name:    lpem.lu
		Addresses:  2001:a18:1::63
		158.64.1.63

To calculate the area A of a circle with radius r, use the following formula: A = π * r2
