Task 1 - Create basic framework

In this task we create the basis for our course. It is important that we keep a clean structure from the beginning. Thus, the content is easier to find for us, as well as for visitors and search engines.

  1. Create a directory in your htsta directory for the first chapter. Naming convention: chapterXX_ABC . XX represents the chapter number and ABC the chapter name. Examples: chapter01_intro, chapter02_hyperlinks, ...
  2. You save the tasks for the respective chapters in the subfolder tasks of the respective chapter.
  3. You save the solutions to the tasks in the solutions of the respective chapter.

Note that all folder and file names contain only lowercase letters and no umlauts or special characters.

1. Folder structure

Create the folder structure for the first chapter as explained above. Copy this file into the subfolder tasks. The structure should be as follows:

htsta/chapter01_intro/tasks/task1.html

The slashes / separate the respective directories. This representation is also used in the navigation bar of the browser.

2. Create Homepage

Now create your first web page in the folder chapter1_intro.

Reference File

We first create a reference file with the basic HTML structure. This file will be the basis for all further web pages we create.

The file should be saved as template.html.

(chapter01_intro/template.html)

To create HTML code we use a simple text editor such as Notepad++.

Homepage

With the help of our reference file we now create our first web page with content. The file is the start page and must therefore be called index.html. Use as title for your web page HTSTA - Program static web pages.

3. Test and validation

Now use an internet browser to view the web page. Also make sure that your HTML code is correct and complies with the standard. You can check this with a validation tool, which is available both as plugins for browsers and online on the internet.
