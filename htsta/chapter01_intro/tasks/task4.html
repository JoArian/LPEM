Task 4 - Transfer to HTML

HTML offers a wealth of tags/elements to display the content according to its meaning. In Word, you can implement this using the various menu items.

However, the principle is the same in both cases. To illustrate this, in this task you will convert a Word document into an HTML file.

Create the following file for this purpose:

Transfer to HTML

Convert the entire contents of this Word document into the HTML file task4.html.
