Indentation

First, take a look at the source code of this page. As you note, the HTML code is not very readable. Edit the code according to the following rules:

  1. Add a line break after each start tag and each end tag.
  2. Split any line of text longer than half the screen with a line break.
  3. Indentate (→ TAB) the entire content between all start and end tags.
