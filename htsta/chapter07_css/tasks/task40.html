Inline CSS

To change the appearance of an HTML element with CSS you can add the style attribute to this element.

For example, to change the CSS style of a paragraph:

<p style="">...</p>

As the value of the style attribute you specify CSS properties and their values. For example, to display the text of the paragraph in red color, use the color property with the value red:

<p style="color: red;"> Paragraph in red font. </p>

Paragraph in red font.

If you want to change several CSS properties, you have to separate them with a semicolon (;). For example, to additionally display the text in bold, set the font-weight property with the value bold:

<p style="color: red; font-weight: bold;"> Paragraph in red bold font. </p>
