Other CSS selectors

So far we have only used the element selector. With the group selector we could additionally group several elements together.

However, this way certain styles can only be applied to all elements of a type (e.g. to all paragraphs).

ID and class selector

The ID and class attribute can be used to address the elements of the HTML document more specifically:

  • ID attribute: defines a unique identifier for an element:
    <p id="introduction">Unique paragraph with ID "introduction".</p>
  • Class attribute: defines one or more class names for an element:
    <p class="reference">Paragraph which belongs to the class "reference".</p>

In CSS, these elements can now be defined with the id und class selectors to apply CSS styles.

To select the element with the ID id-name: 

    
    #id-name {
      CSS properties;
    }

To select all elements of the class class-name: 

    
    .class-name {
      CSS properties;
    }
