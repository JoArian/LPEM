Task 48

Internal styles

So far we have learned two ways to combine CSS rules and HTML elements within an HTML document:

Inline Style
The CSS rules are added directly to the HTML elements with the attribute style.
Internal Style Sheet
The CSS rules are all stored in the <head> in one script element of the HTML file.

External Style Sheet

However, there is a third possibility, which is especially useful if you want to apply the same CSS style to several pages. You combine all CSS rules in an external file and then include this file in the HTML pages:

External Style Sheet
The CSS rules are all stored in a separate file (extension .css), which is referenced in the HTML file with the link element in the <head>.

Use the following website to inform yourself about the External Style Sheet: https://www.w3schools.com/css/css_howto.asp
