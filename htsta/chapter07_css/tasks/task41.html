Task 41

In the previous task we saw how to add CSS style inline in the HTML code to each element. However, this method has some disadvantages:

  • To add the same CSS style to multiple elements of the same type you need to duplicate the code.
  • There is no strict separation between HTML and CSS code.

Internal/Embedded Style Sheet

Instead of specifying the CSS style inline for each element, you can specify the CSS rules all in the <head> of the HTML file. The internal style is specified within the style element in the <head>. 

    
    <head>
      <style>
        <!-- Internal Style Sheet -->
      </style>
    </head>

This method allows you to apply CSS rules to the HTML elements of the entire page. To do this, you use so-called selectors .

CSS selectors

To change the style of an HTML element you have to select the element and then define the CSS properties. 

    
    <head>
      <style>
        Element {
          CSS properties;
        }
      </style>
    </head>
