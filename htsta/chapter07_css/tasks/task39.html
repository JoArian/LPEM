Task 39

The task of CSS is to Design and arrange the elements of a page. Thus, the CSS style can be used to design the presentation of the content of the page.

Apply CSS styles

On the side of the CSS Zen Garden you can apply different CSS styles to the content.

Select some designs on the page to discover the possibilities of CSS. With each design a different CSS style is applied to the page.

Turn off styles

To make it clear that the content is completely independent of the design you can turn off the CSS style using the Web Developer Tools .

CSS style

Turn off the CSS style using the Web Developer Tools:

  • CSS → Disable All Styles

You now see the HTML page, completely without CSS. The visual formatting now corresponds to the standard CSS formatting. This formatting also corresponds to what was previously applied to our HTML code.

Browser style

To see the naked HTML page without any processing, you also need to turn off the browser style:

  • CSS → Disable Browser Default Styles
