Task

For all task points, the CSS properties must be applied inline using the style attribute!

  1. The text of the heading Text template hould be displayed in red color.
  2. Change the appearance of the first paragraph using the style attribute:
    1. The text should be displayed in bold.
    2. The text should also be displayed in italic font.
  3. The second paragraph should be displayed in the same way as the first.
  4. All paragraphs should be displayed like the first paragraph. In addition, the text should also be underlined.
