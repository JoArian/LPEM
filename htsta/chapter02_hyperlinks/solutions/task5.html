Parts and meaning of a URL

  1. The first part of a URL is the Scheme/Protocol, it tells the web which protocol to use. HTTPS (Hypertext Tranfer Protocol Secure) is the most common
  2. The second part is a subdomain, which is basically telling the Web which page to go to like Home, Settings and so on
  3. The third is a second-level Domain which is just the name of the website
  4. The fourt part is a third-level Domain which specifies which type of etry the organization registers on, like .com, .edu, .lu and so on
  5. The fifth part is a subdirectory which tells you more specifically which part of the website youre on, for example if youre on Settings it could say graphics, privacy, general and so on
  6. Lastly theres the Path which specifies the location of the file or resource that the user wants to access.
