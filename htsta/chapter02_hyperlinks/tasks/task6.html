Task 6 - Hyperlinks

A hyperlink represents a reference to a file and usually consists of the following parts:

  1. Start-tag <a> and End-tag </a>
  2. An attribute href which contains the URL of the target file as value
  3. Content of the a element that represents the clickable link in the browser.

Example: <a href="http://www.lpem.lu">Clickable content</a> ergibt Clickable content

If a link is to be opened in a new window/tab, then the attribute target with the value _blank can be added to the a element.
