Task 10 - Link HTML Course

Of course, we want to apply the knowledge we learned in this chapter to our HTML course. This will also make it easier for us to navigate within our course.

Main page

First, we create a main page at the top level of our course:

htsta/index.html

This page should contain at least the following:

  1. A title
  2. A course overview with links to the respective chapters. These links point in each case to the start page of the chapter. e.g. htsta/chap01_intro/index.html
Example of a possible solution.

Chapter overview

The index.html of each chapter should contain the following links:
  1. Link back to main page (htsta/index.html).
  2. Listing with linking of all tasks in the chapter.
Example for chapter 1.

Task linking

  1. Each task should also include a link to the solution.
  2. Likewise, each solution should contain a link to the task.
