Task 7 - Relative and absolute URLs/links

In the previous task we used different types of links. On the one hand we used relative links with which we linked our pages. Then, on the respective pages, we used absolute links to external websites.

So let's go over both types again:

Absolute/Global URLs

An absolute or global link denotes the absolute locality of a file. As already seen in task 5 , the URL is composed of several parts. A global URL can always be inserted on any page and always points to the same destination.

Relative URLs

Relative URLs are a bit more complex. They refer to the location we are currently in. Here are a few examples:
  • ./task7.html - A URL to another page in the same directory. (The ./ at the beginning is optional)
  • ../solutions/ - The two dots (..) mean one level up in the directory structure, then into the directory loesungen. Since we have not specified a file, the search is for index.html.
